Description

Code Composer Studio (CCS) is the integrated development environment for TI's DSPs, microcontrollers and application processors. Find out more about CCS.

CCS for C2000

There are a few versions of CCS available for download or purchase. For C2000, we recommend the following:
- CCS-FREE: Use a full version of CCS which is limited to an XDS100 hardware emulator connection. All controlSUITE-compatible tools include on-board XDS100 emulation.
- CCS-MCU: $445. Full version for support of all TI Microcontrollers: TMS320C2000, MSP430, Stellaris and Cortex R4. Can be used with any emulator (like the XDS500).
